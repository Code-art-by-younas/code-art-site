# CodeArt Site – My Developer Landing Page

A modern, responsive personal landing page built with **HTML, CSS, and JavaScript**.  
It showcases my skills, services, pricing packages, and client testimonials – all in a sleek SaaS-style UI.

## ✨ Features

- Glassmorphism sticky navigation bar  
- Smooth scroll reveal animations (fade, slide up, zoom)  
- Hero section with call-to-action buttons  
- Services / expertise cards with hover glow effects  
- 3 pricing packages (Starter, Pro Business, Elite Custom)  
- Testimonials slider with autoplay and dot navigation  
- Fully responsive – works on mobile, tablet, desktop  
- Footer with social links (LinkedIn, GitHub, Email)

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flex, Grid, custom animations)  
- JavaScript (ES6)  
- Font Awesome icons  
- Google Fonts (Inter)
